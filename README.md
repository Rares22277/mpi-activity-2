# mpi-activity-2
SAT Solver
This is a simple SAT solver implementation in Python using:

Resolution
Davis-Putnam (DP)
Davis-Putnam-Logemann-Loveland (DPLL)
Usage
Each algorithm is in its own file:

resolution.py
DP.py
DPLL.py
CNF formulas are represented as lists of clauses:

cnf_formula = [[1, -3, 4], [-1, 2], [2, -4]]
